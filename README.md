# Основные понятия


* Репозиторий - хранилище вашего проекта с историей изменений

* Коммит - фиксация изменений с уникальным ID

* Ветка (branch) - отдельная линия разработки

* master/main - основная ветка проекта

* origin - стандартное имя удаленного репозитория

* Working Directory - рабочая папка с файлами

* Staging Area - область подготовленных файлов

## Основные команды

git status                  проверить состояние
git add имя_файла           добавить конкретный файл
git add .                   добавить все файлы
git commit -m "сообщение"   зафиксировать изменения
git push                    отправить на сервер
git init                    создать новый репозиторий
git clone URL               склонировать существующий репозиторий
